# Fantasy MMO

This is the starter GitHub Pages site for `space-2d-mmo`.

The project is now aiming at a cozy 2D fantasy MMO: adventurers, little realms, friendly quests, and eventually other players sharing the world.

In Discord, ask the bot to:

- give us three fantasy quests
- make a tiny village demo plan
- summarize the project so far

Maintainers can replace this page with the real demo when the project has one.
